# 📊 Sales Data Analysis & Dashboard Project

## 🔹 Overview
This project focuses on analyzing sales data using Microsoft Excel to generate meaningful business insights and build an interactive dashboard for decision-making.

---

## 🔹 Tools & Technologies Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning Techniques  
- Data Visualization (Bar Chart, Line Chart)  

---

## 🔹 Project Objectives
- Clean and organize raw sales data  
- Automate calculations like Total Sales  
- Analyze sales performance by category and city  
- Identify trends over time (monthly analysis)  
- Present insights using a dashboard  

---

## 🔹 Key Features
- Processed 300+ rows of sales data  
- Created calculated field: **Total Sales (Price × Quantity)**  
- Built Pivot Tables for:
  - Sales by Category  
  - Sales by City  
  - Monthly Sales Trend  
- Designed a professional dashboard layout  
- Identified top-performing category and region
